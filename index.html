Country Rest Api App

this project is concerned with integrating the [REST Countries API](https://restcountries.eu) and to pull country data and display it like in the designs.

where user can: - See all countries from the API on the homepage - Search for a country using an `input` field - Filter countries by region - Click on a country to see more detailed information on a separate page - Click through to the border countries on the detail page - Toggle the color scheme between light and dark mode

Ip Address Tracker

this project is concerned with building out an IP Address Tracker app and display on map using [IP Geolocation API by IPify](https://geo.ipify.org/)

where user can: - View the optimal layout for each page depending on their device's screen size - See hover states for all interactive elements on the page - See their own IP Address on the map on the initial page load - Search for any IP addresses or domains and see the key information and location

Count Down Timer

this project is concerned in building out a countdown timer and get it looking as close to a very best design as possible.

where user can: - See hover states for all interactive elements on the page - See a live countdown timer that ticks down every second

Todo App

this project is concerned with building out a todo app.

where user can: - View the optimal layout for the app depending on their device's screen size - See hover states for all interactive elements on the page - Add new todos to the list - Mark todos as complete - Delete todos from the list - Filter by all/active/complete todos - Clear all completed todos - Toggle light and dark mode - Drag and drop to reorder items on the list

Calculator App
